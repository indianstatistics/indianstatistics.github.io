Bibliographic database

indianstatistics.org uses a bibtex database as a centralised repository of bibliographic information. The entire database is hosted on github for public use. You can clone the repository to get the database.

We would like to invite users to contribute to further developing this database, including by contributing bibliographic information of your work on any aspect of Indian economy and society. Please send us entries in bibtex format for inclusion in the database. Trusted contributors can be given direct access to github repository to push their additions to the repository directly.
